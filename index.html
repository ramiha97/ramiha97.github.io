AI Text Summarization

I developed an AI-powered text summarization system designed to efficiently condense large volumes of text using LLMs and vector-based clustering techniques. The system intelligently reduces the text length before sending it to the model by applying semantic vectorization and K-Means clustering, ensuring that only the most contextually relevant content is summarized. This approach significantly optimized processing time and reduced API costs while maintaining high-quality summaries.

  • ⚙️ Tools & Techniques AWS Bedrock LLM for large-scale text summarization SentenceTransformers for high-dimensional text embeddings Scikit-learn (K-Means) for clustering and vector reduction NumPy & Pandas for data processing and feature management
  • Tech Stack Language: Python AI/ML Frameworks: AWS Bedrock, Transformers, SentenceTransformers Libraries: Scikit-learn, NumPy, Pandas Architecture: Embedding → Clustering → LLM Summarization Pipeline

Face Recognition System

I developed an AI-powered face recognition system that identifies and retrieves matching faces from large datasets with high accuracy and speed. The system extracts facial embeddings using an InsightFace model deployed on a GPU-enabled EC2 instance, then performs nearest-vector searches in a Zilliz vector database to find similar faces. Matched vectors are linked to their corresponding images stored in Amazon S3 and enriched with metadata from MongoDB. All processes are orchestrated through an ECS-hosted API, providing seamless, real-time face identification and data retrieval for clients..

  • Machine Learning & AI: InsightFace, PyTorch Infrastructure: AWS EC2 (GPU), AWS ECS, AWS S3 Databases: Zilliz (Milvus) for vector storage, MongoDB Atlas for metadata Backend Framework: Fastapi (Python)

Generative AI Chatbot

I developed an AI-powered chatbot built on Amazon Bedrock using large language models (LLMs) to deliver intelligent, context-aware interactions. The chatbot is personalized for a specific application through Retrieval-Augmented Generation (RAG) and advanced prompt engineering, enabling it to access domain-specific knowledge and provide accurate, real-time responses. This architecture ensures dynamic, adaptive conversations tailored to each user’s needs while maintaining high scalability and reliability.

  • Tech: AI & NLP: Amazon Bedrock, Large Language Models (LLMs), Prompt Engineering, RAG Backend Framework: Fastaspi (Python) Data Layer: Vector Database (Zilliz) Languages: Python

GIS System Analysis

I developed a GIS-based analytical mapping system capable of visualizing and analyzing millions of geospatial data points in real time. The system leverages K-Means clustering to efficiently group large-scale location data, providing meaningful spatial insights and pattern detection. All data is managed and queried through Amazon RDS PostgreSQL with PostGIS extensions, enabling high-performance geospatial computations and map rendering.

  • Tech: Data Processing & Analytics: K-Means Clustering Database: PostgreSQL with PostGIS (Amazon RDS) Backend Framework: Fastapi (Python) Visualization & Mapping: Leaflet.js, GeoJSON Cloud & Infrastructure: AWS RDS, EC2, S3 Languages: SQL, JavaScript
